What is Ethereum?

Ethereum (ETH) is one of the leading worldwide crypto-currencies, established in 2015 and now standing next to Bitcoin(BTC). The blockchain of the cryptocurrency allows developers to create smart contracts. A huge number of large crypto-companies use this currency.

What is a Smart Contract?

Smart contract is the algorithm stored inside the blockchain of cryptocurrencies. In our case—Ethereum—is number one among the those on which it is possible to create smart contracts. The main purpose of such contracts is the automation of commands from various conditions coming from user inputs.
